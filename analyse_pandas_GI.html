I- Introduction et Dfinitions

1. Quest ce que la bibliothque PANDAS ? Pandas est une bibliothque Python open-source utilise pour l'analyse et la manipulation de donnes. Elle est construite au-dessus de NumPy.

2. Utilit et champs d'application :

  • Utilit : Nettoyage, exploration, transformation et visualisation des donnes.
  • Champs d'application : Finance, Marketing, Logistique (Supply Chain), Science des donnes.

3. Installation :

  • pip install pandas
  • Importation : import pandas as pd

4. Structures de donnes :

  • Series : Unidimensionnelle (1 colonne + index).
  • DataFrame : Bidimensionnelle (Tableau lignes/colonnes). C'est la structure principale.

5. Structure des donnes : Elles doivent tre tabulaires (lignes et colonnes), avec des types de donnes cohrents par colonne.

6. Composition d'un DataFrame : Il est constitu de Data (valeurs), d'un Index (tiquettes des lignes) et de Columns (tiquettes des colonnes).

    subset = df_graph[df_graph['Supplier name'] == supplier].reset_index()
    
    plt.plot(subset.index, subset['Manufacturing costs'], marker='o', label=supplier)

VI- Graphiques et Courbes (Thorie)

4. Citez les types de graphes et fonctions associes :

  • Courbe (Line plot) : df.plot() ou plt.plot(). Utile pour les volutions temporelles.
  • Histogramme (Bar plot) : df.plot(kind='bar') ou plt.bar(). Utile pour les comparaisons.
  • Nuage de points (Scatter plot) : df.plot(kind='scatter') ou plt.scatter(). Utile pour les corrlations.

5. Dessiner des courbes : On utilise gnralement plt.plot(x, y) o x est l'axe horizontal et y l'axe vertical.

6. Comment changer la couleur dune courbe ? On utilise le paramtre color (ou c).

  • Exemple : plt.plot(..., color='red')

7. Comment ajouter deux courbes sur un mme graphe ? Il suffit d'appeler la fonction plt.plot() deux fois de suite dans la mme cellule avant d'afficher le graphe avec plt.show().

8. Comment crer la lgende et la positionner ?

  • Il faut ajouter le paramtre label='Nom' dans plt.plot().
  • Ensuite, on appelle plt.legend() pour l'afficher. On peut utiliser loc='best' pour la positionner automatiquement au meilleur endroit.
